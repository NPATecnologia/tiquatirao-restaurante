# Tiquatirão Restaurante — Site Institucional

Site institucional de restaurante. **Status: concluído.** Manutenção apenas.

## Stack

- **Framework:** Next.js 16.2 (App Router)
- **Linguagem:** TypeScript 5
- **Estilo:** Tailwind 4
- **Animações:** GSAP 3.14 + `@gsap/react` + Lenis 1.3
- **Deploy:** Vercel ou FTP (estático)

## Comandos

```bash
npm run dev    # next dev
npm run build  # next build
npm run lint   # eslint
```

## Convenções

- App Router — sem Pages Router
- GSAP + Lenis para animações de entrada e smooth scroll
- Site estático — sem API routes, sem banco
- Conteúdo hardcoded nos componentes (não há CMS)
- Qualquer mudança de conteúdo = editar o componente diretamente
